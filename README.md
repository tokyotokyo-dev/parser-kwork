# Kwork Parser

Программа следит за аккаунтом на [kwork.ru](https://kwork.ru) и присылает уведомления в Telegram.

## Что умеет

- **Новые сообщения** — проверяет входящие на Kwork и сразу сообщает, если кто-то написал.
- **Статистика кворков** — следит за просмотрами и заказами по каждому кворку. Если что-то изменилось — присылает отчёт.
- **Сводка** — раз в несколько часов показывает общую картину: сколько просмотров, сколько заказов.
- **Команды в Telegram** — можно в любой момент запросить `/stats`, `/inbox` или `/summary`.

## Как это работает

Программа открывает Kwork в фоновом режиме (браузер без окна), заходит в ваш аккаунт и собирает данные. Всё сохраняется в базу данных на вашем компьютере. Уведомления приходят в ваш личный Telegram-бот.

Никакие данные никуда не отправляются — всё остаётся только у вас.

## Что нужно для запуска

1. **Python 3.12 или новее** — на компьютере должен быть установлен Python.
2. **Telegram-бот** — создаётся бесплатно через [@BotFather](https://t.me/BotFather) в Telegram.
3. **Аккаунт на Kwork** — обычный аккаунт продавца.

## Настройка и запуск

### 1. Установка

Скачайте проект, откройте его папку в терминале и выполните:

```
pip install -r requirements.txt
playwright install chromium
```

### 2. Настройка .env

Откройте файл `.env` в любом текстовом редакторе и укажите свои данные:

```
TELEGRAM_BOT_TOKEN=токен_вашего_бота
TELEGRAM_CHAT_ID=ваш_id_чата
HEADLESS=true
```

**Как узнать TELEGRAM_CHAT_ID:**
- Напишите своему боту любое сообщение (например `/start`).
- Откройте в браузере: `https://api.telegram.org/botТОКЕН_БОТА/getUpdates`
- В ответе найдите поле `chat` → `id`. Это число и есть ваш CHAT_ID.

### 3. Первый вход

Запустите программу:

```
python main.py
```

В первый раз откроется окно браузера. Войдите в свой аккаунт Kwork, при необходимости решите капчу. Затем нажмите Enter в терминале — сессия сохранится, и окно закроется.

Все последующие запуски будут работать в фоне, без окна браузера.

### 4. Работа

После первого входа программа:
- проверяет сообщения каждые 3 минуты,
- собирает статистику раз в час,
- присылает сводку каждые 6 часов.

Частоту проверок можно изменить в файле `.env`:

```
MESSAGE_CHECK_INTERVAL=3
STATS_CHECK_INTERVAL=60
SUMMARY_INTERVAL_HOURS=6
```

## Команды в Telegram

Напишите боту в Telegram:

- `/start` — список команд
- `/inbox` — проверить новые сообщения прямо сейчас
- `/stats` — собрать свежую статистику
- `/summary` — показать сводку по всем кворкам

## Если что-то пошло не так

**Сессия сбросилась** — удалите файл `state.json` и запустите программу заново. Откроется окно для повторного входа.

**Бот не отвечает** — убедитесь, что вы написали боту первым. Telegram не разрешает ботам начинать диалог.

**Kwork изменил вёрстку** — если программа перестала находить сообщения или статистику, возможно, Kwork обновил дизайн. В папке проекта появятся скриншоты `debug_inbox.png` и `debug_stats.png` — они помогут понять, что именно сломалось.
